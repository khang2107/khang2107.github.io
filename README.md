# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 Modern and clean design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Smooth scrolling and animations
- 🎯 Interactive navigation
- 💼 Portfolio/Projects showcase
- 📧 Contact form
- 🎭 Smooth transitions and hover effects
- 🌐 Social media integration

## Sections

1. **Hero Section** - Eye-catching introduction with call-to-action buttons
2. **About** - Personal introduction and statistics
3. **Skills** - Technologies and tools showcase
4. **Projects** - Portfolio of completed projects
5. **Experience** - Professional timeline
6. **Contact** - Contact form and information
7. **Footer** - Social links and copyright

## Customization

### Personal Information

Edit the following in `index.html`:

1. **Name and Title** - Update in the hero section:
   ```html
   <h1 class="hero-title">Your Name</h1>
   <h2 class="hero-subtitle">Software Developer & Designer</h2>
   ```

2. **About Section** - Customize your bio and stats
3. **Skills** - Add/remove skills as needed
4. **Projects** - Update project cards with your own work
5. **Experience** - Modify the timeline with your experience
6. **Contact Info** - Update email, phone, and location

### Social Links

Replace placeholder links throughout the file:
- GitHub: `https://github.com/yourusername`
- LinkedIn: `https://linkedin.com/in/yourusername`
- Twitter: `https://twitter.com/yourusername`
- Email: `your.email@example.com`

### Colors

Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... more colors */
}
```

### Images

Replace the placeholder profile image:
1. Add your image to an `images` folder
2. Update the `.image-placeholder` section in the About area
3. Add project screenshots to project cards

## Setup

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content to match your information
4. Deploy to your preferred hosting service
